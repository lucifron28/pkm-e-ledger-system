# Data Sources

This folder may contain local-only official financial report samples used as
layout and requirements references for the PKM e-Ledger System.

Reference files in this folder must not be imported as application data, seed
data, tests, documentation examples, or demo output. They may contain personal
names, student names, signatories, and real financial records.

Only this `README.md` is intended to be committed. Spreadsheet, CSV, and PDF
source files under `data-sources/` are ignored by git.
